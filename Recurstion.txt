재귀함수(Recursive Function) : 자기 자신을 다시 호출하는 함수
=> 재귀 함수의 종료 조건을 반드시 명시해야 함, 종료조건을 제대로 명시하지 않을경우 함수가 무한히 호출될수 있음

예시 : 펙토리얼, 유클리드 호제법

유클리드 호제법 : 최대공약수 계산
=> 두 개의 자연수 A,B에 대해 (A>B) A를 B로 나눈 나머지를 R이라 할때
이때 A와 B의 최대공약수는 B와 R의 최대공약수와 같다
GCD(192,162)(GCD : 최대공약수)
단계	A	B
1	192	162
2 	162	30
3	30	12
4	12	6  => GCD(192,162) = 6이 됨
def gcd(a,b):
    if a%b==0:
          return b
    else:
          return gcd(b, a%b)

print(gcd(192,162))
